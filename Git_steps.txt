Kā strādāt ar Git no VSCode?
1. GitHub vietnē esošu datņu lejupielāde, rediģēšana, izmaiņu fiksēšana un augšupielāde atpakaļ Git Hub serverī.
Atrod interesējošu GitHub repozitoriju, piemēram, https://github.com/startitfonds/HTML-prog-skolotaji-2019-finalversija
Izveido tās zaru savā GitHub kontā - poga Fork GitHub lapā

2. Jaunu versiju kontrolei un augšupielādei GitHub serverī paredzētu datņu izveide ar VSCOde - piemērs.
Izveido jaunu mapi, kurā liksim lietas, kas jāpakļauj versiju kontrolei.
Mapē izveido jaunu failu.
Izpilda darbību View-Command Palette.. (vai CTRL+SHIFT+p) un uzraksta vai izvēlas komandu Initialize Repository un apstiprinam mapi versiju kontrolei.
Kaut ko ieraksta failā, saglabā izmaiņas
Izpilda Stage Changes (+)
Ieraksta kaut ko iekš Message un izpilda Commit (V)
